## 技术栈参考

- **UI 框架**: Tamagui
- **路由**: Expo Router
- **状态管理**: React Context 或 Zustand
- **存储**: Expo SecureStore / AsyncStorage
- **日期处理**: date-fns 或 dayjs
- **图表（如需要）**: react-native-chart-kit（极简配置）或 Tamagui 内置图表

---

## 设计参考原则（开发时需时刻提醒）

1. **平静、中性、不评判** - 所有文案和视觉元素
2. **无游戏化** - 无打卡、无连续记录、无状态色
3. **个人笔记感** - 而非健身教练
4. **长期使用友好** - 不制造健康焦虑
5. **极简视觉** - iOS 原生体验、柔和字体、宽松留白
6. **低对比度** - 克制的配色
7. **微交互** - 细腻但克制，无炫技
